# 🌐 Web Development Assignments – Raj Sharma

This repository contains **6 web development assignments** completed as part of my web development course.
Each project focuses on building different types of web pages using **HTML, CSS, and basic JavaScript** to practice design, layout, and functionality.

---

## 🗂 Projects Overview

### 1️⃣ Class 5 Timetable

A clean and visually appealing **weekly timetable** for Class 5.
**Features:**

* Subjects listed Monday to Friday
* Modern glassmorphism design
* Hover effects on table cells
* Responsive layout

### 2️⃣ 50 Richest People List

A **dynamic list of the 50 richest people in the world**.
**Features:**

* Name, net worth, country, and ranking
* Table layout with hover highlights
* Clean, readable design

### 3️⃣ ByteBox – GNIOT Canteen Menu

A **canteen menu webpage** for GNIOT College.
**Features:**

* Menu items with images, description, and price
* Responsive grid layout
* Hover effects on menu cards
* Gradient background with blurred overlay

### 4️⃣ Countries, Capitals, Currency & Flags

A **list of countries** with their capital cities, currency, and national flags.
**Features:**

* Table or card layout
* Displays country flag, capital, and currency
* Easy-to-read structured design

### 5️⃣ 💍 JeevanSaathi Match Form

A **simple matchmaking form** inspired by matrimonial websites.
**Features:**

* Collects gender, age, religion, mother tongue, and location
* Form submission via email
* Clean, centered card design
* Hover effect on submit button

### 6️⃣ Flipkart Frontpage Clone

A **clone of Flipkart’s homepage** built for practice.
**Features:**

* Header with search bar and navigation
* Product cards with images, titles, and prices
* Responsive grid layout
* Focus on layout and design accuracy

### 7️⃣ Student Admission Form

A **form for student admission** demonstrating form handling and layout skills.  
**Features:**
* Collects student name, age, class, parent info, and contact details
* Validation using basic JavaScript
* Clean and structured form layout

### 8️⃣ Amazon Homepage Layout

A **practice layout inspired by Amazon’s homepage**.  
**Features:**
* Navigation bar with logo, search bar, and menu items
* Multiple product sections (Electronics, Fashion, Deals, Home Essentials)
* Product cards with images, titles, and prices
* Hover effects and responsive design
---

## 🛠 Technologies Used

* **HTML5** – structure and content
* **CSS3** – styling, layout, and responsive design
* **JavaScript (optional)** – minor interactivity in some projects
