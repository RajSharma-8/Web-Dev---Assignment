# 🌎 Top 50 Richest People in the World

This project displays a visually appealing table of the top 50 richest people in the world, including their rank, photo, name, net worth, and country.

---

## Features

- **Responsive Design** – Works on all screen sizes.
- **Hover Effects** – Highlights rows when hovered.
- **Gradient Headers** – Stylish gradient text for the main heading.
- **Profile Photos** – Circular images for each individual.
- **Rank Colors** – Top 3 ranks highlighted in gold, silver, and bronze.

---

## Technologies Used

- **HTML5** – Structure of the webpage.
- **CSS3** – Styling, gradients, hover effects, and responsive design.
- **Fonts** – Uses `'Poppins', sans-serif` for a modern look.

---

## How to Use

1. Clone or download the repository.
2. Ensure the `assets/images` folder contains all profile pictures.
3. Open `index.html` in any modern web browser.
