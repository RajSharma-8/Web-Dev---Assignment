# Student Admission Form Webpage

## Overview
This project is a **Student Admission Form** built with **HTML** and **CSS**, designed for universities or colleges to collect student admission details. The form features a modern layout, an animated gradient background, and a full-page background image, providing an attractive and user-friendly interface.

---

## Features
- Collects **12 essential student details**:
  - Full Name
  - Age
  - Gender
  - Date of Birth
  - Email Address
  - Contact Number
  - Address
  - Course Selection
  - Previous Qualification
  - Percentage / Marks
  - Upload Photo
- **Animated Gradient Background** inside the form for a modern dynamic look.
- **Full-page Background Image** to enhance visual appeal.
- **Responsive Design**: Works seamlessly on mobile, tablet, and desktop.
- **User-friendly Inputs**: Proper labels, placeholders, and input types (`text`, `email`, `number`, `date`, `radio`, `select`, `file`).
- **Hover Effects** on the submit button.

---

## Technologies Used
- **HTML5**: For the form structure.
- **CSS3**: For styling, animated gradient background, and responsive layout.
- **Responsive Design**: Ensures accessibility across devices.

---

## How to Use
1. Clone or download the repository.
2. Ensure the following assets are available:
   - `assets/background.jpg` → Background image.
   - `assets/favicon.png` → Favicon.
3. Open `index.html` in a web browser.
4. Fill out the form with required student details.
5. Click the **Submit** button (currently configured to `mailto:admissions@university.edu`).

---

## Customization
- **Background Image**: Replace `assets/background.jpg` with your preferred image.
- **Gradient Colors**: Modify the `linear-gradient` values in `.admission-form` CSS.
- **Form Action**: Change the `action` attribute in the `<form>` tag to submit to a backend server or database.
- **Courses**: Add or remove `<option>` tags in the course selection dropdown.
