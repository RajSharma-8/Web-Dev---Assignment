# 🍔 ByteBox – GNIOT College Canteen Menu

A modern and visually appealing **canteen menu webpage** for **GNIOT College**, built using **HTML and CSS**.
This project showcases menu items with images, descriptions, and prices in a **responsive card layout**.

---

## 🚀 Features

* 🍟 Display of popular canteen items like sandwiches, burgers, fries, pizza, pasta, Maggi, coffee, tea, and shakes.
* 🖼️ Image-based menu cards with hover effects.
* 💳 Clear display of item prices.
* 🎨 Colorful, gradient background with blurred overlay.
* 📱 Fully responsive layout using CSS grid.
* 🖱️ Smooth hover animations for interactive UI.

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
